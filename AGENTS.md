# AGENTS.md - Journal Submodule

## Overview
This is the QuantTide Founder Journal submodule, containing daily journal entries.

## Structure
- `default/` - Default journal templates
- `product/` - Product-related journal entries
- `work/` - Work-related journal entries
- `YYYY-MM-DD.md` - Daily journal entry files

## Commands
- `npm run journal` - Create new journal entry
