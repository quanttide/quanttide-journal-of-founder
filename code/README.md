# 编程日志

Vibe Coding（直觉编程）的数字化管理系统，记录 AI 辅助编程的工作流程。

## 工作流

```
decision    →  plan    →  report    →  evaluation    →  retrospective
决策           计划       报告         评估              回顾
```

## 文件夹

按项目分类，每个子文件夹对应一个项目：

- `thera/` - thera 平台
- `qtadmin/` - qtadmin 项目

## 文件命名规则

- 格式：`YYYY-MM-DD.md`
- 一天一个日志文件
