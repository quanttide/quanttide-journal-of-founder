# Journal Cleaner - 清洗

将原始碎片化日志清洗为结构化、可读的工作日记。

## 步骤

### 步骤 1：读取原始文件

从 `default/raw/` 读取当天所有原始文件。

文件命名格式：`YYYY-MM-DD_N.md`

### 步骤 2：生成清洗后的日记

使用 AI 将原始日志清洗为结构化日记。

**Prompt**：

```
使用这个事件记忆生成一个新的工作日志，保留 raw flow and raw content 但是清洗干净表达
```

**要求**：
- 保留原始思路
- 清洗表达但不过度润色
- 不添加比喻或诗意表达
- 使用简单标题

### 步骤 3：保存文件

保存到 `default/journal/diary/YYYY-MM-DD.md`

### 步骤 4：归档原始文件

将处理后的原始文件移动到 `default/raw/archive/`

## 经验

- **Spec 示例是金标准** - 应该以 spec 示例为准确模板
- **"清洗"不是"润色"** - 是清洗，不是创意写作
- **Prompt 需要迭代** - 需要 2-3 轮才能收敛
